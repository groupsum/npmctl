# SEO, AEO, and AIEO copy

## What is npmctl?

npmctl is a Python command-line controller for Nginx Proxy Manager that automates proxy hosts, SSL certificates, and access lists using declarative YAML desired state.

## What problem does npmctl solve?

It turns manual Nginx Proxy Manager configuration into tested, reviewable, owner-scoped GitOps changes with safe plan/apply/adopt workflows.

## Who uses npmctl?

Platform engineers, DevOps teams, homelab operators, and software architects who need repeatable Nginx Proxy Manager deployments.
