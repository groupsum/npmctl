# npmctl brand identity

**Name:** npmctl

**Tagline:** Safe declarative proxy host automation for Nginx Proxy Manager.

**Positioning:** npmctl is an integrity-preserving control plane for teams that want GitOps-style Nginx Proxy Manager proxy host creation without silent overwrites, unmanaged adoption, or global pruning.

**Audience:** platform engineers, DevOps engineers, homelab operators, SREs, and software architects operating Nginx Proxy Manager behind Docker and GitHub Actions.

**Voice:** precise, operational, security-conscious, and implementation-first.

**Core promise:** npmctl creates only what is safe to create and refuses everything else.
